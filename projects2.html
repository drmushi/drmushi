Open Educational Resources(OER) Facilitation

Ms.Mushi is one of the co-developers and facilitators of Working with Open Educational Resources module at the Open University of Tanzania.
The module is a component of Digital Fluency course for academics which has been developed in collaboration with OER Africa.
The course is run for four weeks covering topics such as OER Concepts, Creative Commons Licensing, Reusing OER and OER Production. Ms. Mushi was featured as one of the Humans of OER, a campaign that is run by the Open Education Consortium.

TusomeInnovations

TusomeInnovations is a startup that provides access to digital learning resources to children with hearing disability in Tanzania.
Ms. Mushi founded TusomeInnovations in 2017 and through her leadership she has been able to lead the digitization of sign language dictionary that was initially developed by Tanzatoto Foundation.


Online Course Development

Having worked as an educational technologist, Ms. Mushi has been exposed to a number of tools that she has found intriguing and useful in developing robust online courses. The list below contains tools that Doreen recommends for the best experience.

Kahoot!

This is an interactive game-based quiz tool that makes learning fun and engaging. The best part about Kahoot is that it is user-friendly and it can run on multiple platforms

Padlet

Padlet is an excellent tool that can be used by learners of all stages. It provides a creative way of collaborating and displaying content.
It is easy to use and it supports all kinds of media resources.

Articulate Storyline

This is an excellent software for authoring e-learning content and anything that has to do with online courses.
