Educational Technology Tools

Having worked as an educational technologist, I have been exposed to a number of tools that I have found intriguing and helpful.
I believe these tools have enhanced my instructional experience especially in developing robust learning resources

Kahoot!

This is an interactive game-based quiz tool that makes learning fun and engaging. The best part about Kahoot is that it is user-friendly and it can run on multiple platforms

Padlet

Padlet is an excellent tool that can be used by learners of all stages. It provides a creative way of collaborating and displaying content.
It is easy to use and it supports all kinds of media resources.

Articulate Storyline

This is an excellent software for authoring e-learning content and anything that has to do with online courses.
